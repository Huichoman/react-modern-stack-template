### 🚀 Quick Start

1. **Click "Use this template"** to create your repository
2. **Clone your new repository:**
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   cd your-repo-name
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start developing:**
   ```bash
   npm run dev
   ```
